#### Description

Adds the unit to the unassigned WSUS group, letting the unit be linked to (another) apropriate dynamic WSUS group.

### Syntax

:vb: CMS_AddUnitToWSUSUnAssignedGroup() As Boolean

:ps: \[bool\]CMS_AddUnitToWSUSUnAssignedGroup

### Parameters

none

### Return value

:vb: Boolean, TRUE if function completed successfully.

:ps: Boolean, TRUE if function completed successfully.

### Example

##### :vb: **VBScript**
