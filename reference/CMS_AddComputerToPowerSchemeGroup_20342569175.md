### Description

Adds the specified computer to the specified power scheme group

### Syntax

:vb: CMS_AddComputerToPowerSchemeGroup(sUnit, sGroup) as Boolean

:ps: \[bool\]CMS_AddComputerToPowerSchemeGroup -group \<string\>

### Parameters

*sUnit (String)*

Unit name

*sGroup (String)*

Group name

### Return value

:vb: Boolean, TRUE if function completed successfully.
