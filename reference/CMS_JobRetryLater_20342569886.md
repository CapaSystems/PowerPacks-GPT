### Description

Cancels the job execution and will be run at next Agent cycle.

**Note:**Â This function will set bStatus to False in order to skip the Job execution

### Syntax

:vb: CMS_JobRetryLater(sTxt) As Boolean

### Parameters

*sTxt (String)*

Append message to Log

### Return value

:vb: Boolean, TRUE if function completed successfully.

### Example

##### :vb: **VBScript**

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">
