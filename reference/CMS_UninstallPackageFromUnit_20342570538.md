### Description

Uninstalls package from unit

### Syntax

:vb: CMS_UninstallPackageFromUnit(sPackage, sVersion) As Boolean

:ps: \[bool\]CMS_UninstallPackageFromUnit -package \<string\> -version \<string\>

### Parameters

*sPackage (String)*

Name of package

*sVersion (String)*

Version of package

### Return value

:vb: Boolean, TRUE if function completed successfully.

:ps: Boolean, TRUE if function completed successfully.

### Example

##### :vb: **VBScript**

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
If bStatus Then bStatus = CMS_UninstallPackageFromUnit("CapaInstaller", "v1.0")
```

</div>

</div>

##### :ps: **PowerShell**

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
if (CMS_UninstallPackageFromUnit -package 'Pacman for Windows 1.1' -version 'v1.0') {
  $cs.Job_WriteLog("Uninstall", "Pacman for Windows 1.1 will now be uninstalled")
}
```
